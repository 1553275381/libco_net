智能指针在 c++ 中的使用：

1. 一个合格的 c++ 程序在使用指针的时候，不应该在程序代码里看到任何关于 raw pointer(裸指针) 的引用，因为暴露指针这个做法非常危险，即使在你的 class 类的设计里你已经做好了相应的成员或方法的控制权限（比如: public, private, protected），但使用者对指针的管理不当，就会造成悬挂指针，或野指针。

2. 如果你非要使用指针，请随时想起智能指针。
eg:
	T* p = new T(); 			//wrong usage -_-
	
	smart_ptr<T*> sp(new T())   //right usage ^_^
